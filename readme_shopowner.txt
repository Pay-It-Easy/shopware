Modul: "Pay It Easy" Zahlungsmodul

Version: 1.2.0

Beschreibung: Zahlungsmodul zur Durchführung einer Zahlung über das "Pay It Easy" System

Getestet mit Shop-System: Shopware 4.0.7, 4.1.3, 4.2.1

1	Installation
1.1 Installation des Plug-Ins
	a) Laden Sie die Installationsdatei "shopware_Pay-It-Easy_1.2.0.zip" auf dem Shop-Server in ein temporäres Verzeichnis (/tmp) hoch.
	b) Entpacken Sie die Datei in das /tmp Verzeichnis (unzip shopware_Pay-It-Easy_1.2.0.zip).
	c) Kopieren Sie den Inhalt des Verzeichnisses "copy_this" in das Root-Verzeichnis des Shops.
	d) Stellen Sie sicher, dass die neu erstellten Dateien und Verzeichnisse für den Shop lesbar/ausführbar sind (Berechtigung 755).
1.2	Aktivierung des Zahlungsmoduls
	a) Öffnen Sie im Browser die Administrationsansicht des Shops und melden Sie sich als Administrator an.
	b) Im Hauptmenü klicken Sie auf Einstellungen > Plugin Manager und dann auf "Lokale Erweiterungen"
	c) Wählen Sie das Modul "Pay It Easy" aus und klicken Sie auf (+) "Plugin-Installieren".
	d) Klicken Sie auf "Plugin aktivieren".
	e) Im Hauptmenü klicken Sie auf "Einstellungen > Zahlungsarten"
	f) Klicken Sie auf "Pay It Easy xxx", dabei steht "xxx" für die Zahlungsart, die Sie aktivieren möchten
	g) Klicken Sie auf das Häkchen "Aktiv" und anschließend auf "Speichern"
	h) Stellen Sie sicher, dass die Zahlungsart "Pay It Easy Base" deaktiviert ist.

2 Konfiguration
2.1 Konfiguration des Plug-Ins im Shopsystem
	a) Öffnen Sie im Browser die Administrationsansicht des Shops und melden Sie sich als Administrator an.
	b) Im Hauptmenü klicken Sie auf Einstellungen > Plugin Manager und dann auf "Lokale Erweiterungen"
	c) Klicken Sie auf das Plugin "Pay It Easy"
	d) Tragen Sie die Ihnen zugewiesene Händlerkennung in das Feld "Händlerkennung" ein.
	e) Tragen Sie den Ihnen zugewiesenen MAC-Schlüssel in das Feld "MAC-Schlüssel" ein.
	f) Klicken Sie auf den Button "Speichern".
